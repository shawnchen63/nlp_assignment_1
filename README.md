## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install the [requirements.txt](SourceCode/requirements.txt) file.

```bash
cd SourceCode/
pip install -r requirements.txt
```
or using conda:
```bash
cd SourceCode/
conda install --file requirements.txt
```
